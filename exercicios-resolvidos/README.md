# Exercícios Resolvidos

Esta pasta reúne exercícios resolvidos em Visualg, organizados por conceito de lógica de programação.

O objetivo não é apenas apresentar respostas, mas demonstrar:
- raciocínio lógico
- organização de código
- uso consciente das estruturas da linguagem

## Estrutura da pasta

Os exercícios estão separados em subpastas conforme o conteúdo estudado:

- `entrada-saida/` → leitura e exibição de dados
- `condicionais/` → estruturas `Se/Senão` e `Escolha`
- `repeticao/` → laços `Enquanto`, `Repita` e `Para`

Cada subpasta possui exercícios independentes, focados no conceito específico.
